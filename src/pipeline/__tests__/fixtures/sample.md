# Getting Started with TypeScript

TypeScript adds static type checking to JavaScript, catching errors before runtime.

## Why TypeScript?

- **Type Safety**: Catch bugs at compile time rather than runtime
- **Better Tooling**: Autocomplete, refactoring, and navigation work better with types
- **Documentation**: Types serve as living documentation for your code

## Basic Types

TypeScript provides several basic types including `string`, `number`, `boolean`,
`array`, and `object`. You can also define custom types using interfaces and type aliases.

## Getting Started

Install TypeScript via npm: `npm install -g typescript`. Create a `tsconfig.json`
file to configure your project, then compile with `tsc`.
